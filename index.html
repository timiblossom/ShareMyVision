Welcome to SMV project.

Instragram like server component project. It has a web component written in Jruby. It has several separatedly deployable server components: user provision, mobile API, file upload/download, image processing service, email service, etc. The whole thing is an SOA.

Web component

Using Rails and Jruby to display users' uploaded images

Driver/aggregation component

This is the heart of this SOA. This is where Mobiles and Web components send API requests. This is a stateless component which makes calls (sync/async) to other internal services.

File Service

A wrapper over S3 to serve file upload/download and generating encrypted, short-time-to-live urls

Image processing

Generate thumbnails and sizes for an image. Also this could be used to do other image processing.

Email Service

Use to email out different emails with different templates

Mobile API

Written in Jruby with an aggregator component to split out a request into several internal requests and merge the results before returning to the mobile clients

Catalog Service

Light component to manage product catalog
